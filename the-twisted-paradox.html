VECTOR ICONS, 2D

The Twisted Paradox

ABOUT THE PROJECT

The Twisted Paradox is a visual design project developed as a team around the theme of Myth & Folklore. The project focuses on creating a cohesive set of 12 vector icons inspired by characters from Alice in Wonderland, supported by a consistent visual system and layout presentation.

PROJECT TYPE
